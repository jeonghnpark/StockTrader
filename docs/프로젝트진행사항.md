# 주식 포트폴리오 트래커 - 1차 개발 진행 사항

## 1. 프로젝트 개요
* **목적:** 개인적인 목적으로 주식 매매 내역을 기록하고, 포트폴리오의 현재 상태(평가손익, 비중 등)를 한눈에 파악하기 위한 대시보드 구축
* **기술 스택:** Python, Streamlit (UI), Pandas (데이터 처리), yfinance (해외 주식 및 환율 데이터), urllib/re (국내 주식 한글명 크롤링), Plotly (차트 시각화)
* **데이터 저장소:** 로컬 CSV 파일 (`data/trade_history.csv`)

## 2. 주요 구현 기능

### 2.1. 매매 내역 관리 (CRUD)
* **내역 추가:** 사이드바를 통해 매매일자, 계좌, 종목코드, 매매종류(매수/매도), 수량, 단가, 통화 입력.
  * 종목 코드 입력 시 실시간 현재가 및 종목명 자동 조회 후 단가 기본값으로 설정.
  * 수량 증감 단위 최적화 (기본값 1, 1씩 증감).
* **내역 조회:** 전체 매매 내역을 테이블 형태로 제공.
* **내역 수정 및 삭제:** 테이블의 행(Row)을 클릭하면 즉시 편집 폼이 활성화되어 직관적인 수정 및 삭제 가능.

### 2.2. 계좌 및 통화 관리
* **다중 계좌 지원:** 여러 계좌를 등록하고 관리할 수 있으며, 메인 화면에서 특정 계좌만 필터링하여 포트폴리오 요약 확인 가능.
* **계좌별 통화 지정:** 각 계좌마다 기본 통화(KRW, USD 등)를 지정(상수화/매핑)하여, 매매 입력 및 수정 시 통화가 일치하지 않으면 에러를 발생시켜 데이터 정합성 유지.
* **원화 환산 모드:** 실시간 USD/KRW 환율을 연동하여, 달러 자산을 포함한 전체 포트폴리오를 원화(KRW) 기준으로 환산하여 볼 수 있는 토글 기능 제공.

### 2.3. 포트폴리오 요약 및 시각화
* **핵심 지표 (Metrics):** 총 자산 평가액, 총 평가 손익(수익률 포함), 총 실현 손익을 상단에 배치.
* **보유 종목 현황:** 현재 보유 중인 종목들의 평균단가, 현재가, 평가금액, 평가손익, 수익률, 비중을 테이블로 제공. (소수점 버림 등 깔끔한 숫자 포맷팅 적용)
* **자산 비중 차트:** Plotly를 활용하여 종목별 자산 비중을 파이 차트로 시각화.

### 2.4. 국내/해외 주식 완벽 지원
* **해외 주식:** `yfinance`를 통해 티커 기반 데이터 및 영문 종목명 조회.
* **국내 주식:** `.KS`(코스피), `.KQ`(코스닥) 접미사를 활용하여 가격을 조회하고, 네이버 금융 크롤링(utf-8 인코딩 적용)을 통해 정확한 한글 종목명 파싱 및 표시.

## 3. UI/UX 개선
* 전체 인터페이스 한글화 완료.
* 타이틀 및 서브타이틀 폰트 크기 조정으로 화면 가독성 향상.
* 종목명 캐싱(Caching)을 통한 API 호출 최소화 및 로딩 속도 개선.
